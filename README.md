
BKLN
====

## what is BKLN?

BKLN is a node.js tiny url creator with some really sweet modules and unit tests under the hood

### modules

- **express**
- **mongoose**
- _expresso_
- node-mongodb-native
- nodemon
- ejs

### installation

```bash
$ git clone git@github.com:forsurerad/BKLN.git
$ cd BKLN
$ npm install
$ nodemon index.js
```
